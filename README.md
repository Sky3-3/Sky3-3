## 🎯 Índice de Proyectos y Repositorios

### 🐍 Fundamentos de Programación y Estructuras en Python

* 📑 **python-condicionales-seguro-medico**
  * 🔧 *¿Qué hace?* Un script enfocado en la implementación de estructuras de control condicional (`if`/`else`) y la anidación de funciones modulares para evaluar perfiles de riesgo en salud.
  * 🛠️ *Tecnologías:* Python (Flujo de Control).

* 📦 **python-estructuras-listas-zip**
  * 🔧 *¿Qué hace?* Un proyecto dedicado a la manipulación de colecciones lineales mutables, emparejamiento de datos paralelos mediante `zip()` y auditoría de costos reales frente a estimados.
  * 🛠️ *Tecnologías:* Python (Listas y Tuplas).

* ✂️ **python-slicing-ordenamiento-listas**
  * 🔧 *¿Qué hace?* Desarrollo algorítmico enfocado en el ordenamiento interno (*in-place*) y segmentación de datos mediante técnicas de rebanado avanzado (*Slicing*) con índices positivos y negativos.
  * 🛠️ *Tecnologías:* Python (Manejo de Arreglos).

* 🔁 **python-bucles-list-comprehension**
  * 🔧 *¿Qué hace?* Modelado de flujos iterativos eficientes mediante bucles `while` de acumulación, ciclos `for` indexados por rangos y optimización de vectores con *List Comprehension*.
  * 🛠️ *Tecnologías:* Python (Estructuras de Repetición).

* 🔐 **python-caesar-vigenere-ciphers**
  * 🔧 *¿Qué hace?* Módulos lógicos de criptografía simétrica para la codificación, decodificación y rotación de caracteres usando aritmética modular y ataques por fuerza bruta (*Grid Search*).
  * 🛠️ *Tecnologías:* Python (Criptografía Clásica).

### 🛠️ Ingeniería, Limpieza y Análisis de Datos (Data Science)

* 🧹 **python-data-cleaning-strings**
  * 🔧 *¿Qué hace?* Pipeline de limpieza de datos (*Data Cleaning*) encargado de transformar bloques de texto plano desestructurados en matrices limpias mediante métodos como `.split()` y `.strip()`.
  * 🛠️ *Tecnologías:* Python (Data Stripping).

* 📁 **python-file-parsing-security**
  * 🔧 *¿Qué hace?* Automatización de flujos de persistencia y transferencia de datos mediante la lectura de estructuras tabulares `CSV`, escritura de texto plano y serialización en formato `JSON`.
  * 🛠️ *Tecnologías:* Python I/O (Sistemas de Archivos).

* 📐 **python-linear-regression-algorithm**
  * 🔧 *¿Qué hace?* Implementación matemática desde cero de una función de pérdida absoluta (*L1 Loss*) y optimización paramétrica por fuerza bruta para calcular rectas de mejor ajuste.
  * 🛠️ *Tecnologías:* Python (Modelos Analíticos).

* 🌿 **pandas-inventory-data-analysis**
  * 🔧 *¿Qué hace?* Análisis descriptivo corporativo y valuación financiera de inventarios comerciales aplicando funciones vectoriales anónimas `lambda` sobre DataFrames de Pandas.
  * 🛠️ *Tecnologías:* Pandas (Ingeniería de Atributos).

* 🛒 **pandas-conversion-funnel-analysis**
  * 🔧 *¿Qué hace?* Modelado transaccional de un embudo de conversión en comercio electrónico a través de uniones por la izquierda (*Left Joins*) y cálculo de deltas de tiempo (*Timedeltas*).
  * 🛠️ *Tecnologías:* Pandas (Análisis Comercial).

* 🧪 **pandas-ab-testing-shoefly**
  * 🔧 *¿Qué hace?* Enfoque experimental enfocado en la evaluación de rendimiento de campañas publicitarias mediante pruebas A/B, cálculo de tasas de clics (CTR) y reestructuración de matrices con tablas pivote.
  * 🛠️ *Tecnologías:* Pandas (A/B Testing / Pivots).

### 📊 Estadística Descriptiva, Inferencial y Visualización

* 🧠 **pandas-jeopardy-data-analysis**
  * 🔧 *¿Qué hace?* Filtrado avanzado de texto mediante expresiones regulares complejas (Regex), rotación de niveles lógicos con `.unstack()` y diseño de un motor estocástico de autoevaluación.
  * 🛠️ *Tecnologías:* Pandas / Regex / Matrices.

* 🏥 **pandas-eda-diabetes-cleaning**
  * 🔧 *¿Qué hace?* Análisis Exploratorio de Datos (EDA) clínico enfocado en la detección e imputación de ceros biológicos incoherentes mediante marcas `np.nan` y recasteo de tipos de datos.
  * 🛠️ *Tecnologías:* Pandas / NumPy (EDA).

* 🎓 **pandas-data-visualization-uci**
  * 🔧 *¿Qué hace?* Resumen analítico univariado de rendimiento académico mediante medidas de tendencia central y dispersión, exportando diagramas de cajas (*Boxplots*) e histogramas de frecuencias.
  * 🛠️ *Tecnologías:* Seaborn / Matplotlib (Estadística).

* 🍋 **matplotlib-sublime-limes-charts**
  * 🔧 *¿Qué hace?* Diseño y configuración de tableros de control visual de series temporales mediante el despliegue coordenado de subtramas múltiples (`subplots`) independientes.
  * 🛠️ *Tecnologías:* Matplotlib (Dashboards).

* ✈️ **pandas-airline-data-analysis**
  * 🔧 *¿Qué hace?* Exploración estadística multivariante de tarifas aéreas mitigando el fenómeno de sobregraficación (*overplotting*) mediante submuestreo aleatorio, opacidad y variables jitter.
  * 🛠️ *Tecnologías:* Seaborn (Análisis Multivariante).

* 🗺️ **pandas-us-census-data-cleaning**
  * 🔧 *¿Qué hace?* Ingesta de datos censales distribuidos por lotes mediante `glob.glob`, imputación deductiva por diferencias lógicas de columnas y manipulación atómica con `.iat[]`.
  * 🛠️ *Tecnologías:* Pandas / Glob (Ingeniería de Datos).

* 🏀 **pandas-nba-statistical-trends**
  * 🔧 *¿Qué hace?* Análisis inferencial de tendencias deportivas evaluando covarianza, correlación lineal de Pearson y ejecución de pruebas de hipótesis de Chi-cuadrado para independencia de factores.
  * 🛠️ *Tecnologías:* SciPy / Pandas (Estadística Inferencial).

* 🏢 **pandas-housing-central-tendency**
  * 🔧 *¿Qué hace?* Diagnóstico comparativo de asimetrías y sesgos positivos en los precios del mercado inmobiliario de Nueva York utilizando funciones estadísticas centrales robustas.
  * 🛠️ *Tecnologías:* SciPy / NumPy / Pandas.

* 🎲 **scipy-poisson-probability-sim**
  * 🔧 *¿Qué hace?* Modelado probabilístico y simulación de control de calidad industrial calculando funciones de masa (`pmf`), acumuladas (`cdf`) e inversas (`ppf`) de la distribución de Poisson.
  * 🛠️ *Tecnologías:* SciPy (Simulación Estocástica).
